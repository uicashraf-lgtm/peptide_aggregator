# Project Instructions

## Version Bump on PR

Before creating any pull request, always bump the plugin version in `peptide_aggregator/peptide-aggregator.php`:
- Increment the patch version (e.g. `0.2.52` → `0.2.53`)
- Update both the `Version:` header comment and the `PA_PLUGIN_VERSION` constant
- Commit the version bump as part of the PR branch before pushing
