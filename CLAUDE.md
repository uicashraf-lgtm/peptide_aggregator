# Project Instructions

## Version Bump on Every Change

Before committing any change, always bump the plugin version in `peptide_aggregator/peptide-aggregator.php`:
- Increment the patch version on both the `Version:` header comment and the `PA_PLUGIN_VERSION` constant (e.g. `0.2.52` → `0.2.53` and `0.3.10` → `0.3.11`)
- Include the version bump in the same commit as the change
